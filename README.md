# ❤️ Heart Attack Risk Prediction using Machine Learning

## 📌 Project Overview

This project predicts heart attack risk using Machine Learning and Logistic Regression. The model was trained on a dataset of over **50,000 patient records** to identify individuals at high risk based on demographic, lifestyle, and clinical factors.

## 🚀 Tech Stack

- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Jupyter Notebook

## 📊 Dataset

- Source: Kaggle
- Records: 50,000+
- Features: 26

## 📈 Project Workflow

1. Data Collection
2. Data Cleaning
3. Exploratory Data Analysis (EDA)
4. Feature Engineering
5. Logistic Regression
6. Model Evaluation

## 📌 Model Performance

- Accuracy: **86.69%**
- Precision: **0.88**
- Recall: **0.87**
- F1 Score: **0.87**

## 🔍 Key Insights

- Diabetes is one of the strongest predictors of heart attack risk.
- High cholesterol and BMI significantly increase risk.
- Logistic Regression performed well with balanced precision and recall.
